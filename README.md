# Restaurant App

Starter Flask application scaffold for a restaurant ordering system.

## Structure

- `backend/app.py` application entry point
- `backend/config.py` Flask configuration
- `backend/models.py` database models
- `backend/routes/` route blueprints
- `backend/static/` CSS, JavaScript, and uploads
- `backend/templates/` Jinja templates

## Run

1. Create and activate a Python environment.
2. Install dependencies from `backend/requirements.txt`.
3. Start the app with `python backend/app.py`.
